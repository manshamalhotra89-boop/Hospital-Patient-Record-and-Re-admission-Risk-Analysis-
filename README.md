# Hospital-Patient-Record-and-Re-admission-Risk-Analysis-
Power BI dashboard for hospital readmission analysis, highlighting patient trends, diagnosis distribution, and risk factors.

**Objective**
To analyze hospital patient data and identify patterns related to patient admissions, diagnosis distribution, treatment types, and factors influencing hospital readmission risk.
The goal of this dashboard is to provide interactive insights that help understand patient trends, risk factors, and healthcare resource utilization using Power BI.

**Key Insights / Features**
Built an interactive Power BI dashboard to analyze hospital readmission patterns.
Analyzed patient admissions trend over time to understand seasonal and yearly patterns.
Identified diagnosis categories with higher patient volumes.
Examined the relationship between length of hospital stay and readmission risk using scatter analysis.
Compared readmission patterns across different regions.
Analyzed treatment types used for different diagnoses.
Evaluated average medications and hospital stay duration for different medical conditions.
Implemented interactive slicers (Year, Region, Gender, Season) for dynamic data exploration.

**Tools Used**
Power BI
Data Visualization
Data Modeling

**Dashboard Pages**

**Hospital Overview**
Patient distribution by diagnosis
Insurance distribution
Monthly admission trends

**Readmission Risk Analysis**
Relationship between length of stay and readmission risk
Average risk score by diagnosis
Regional readmission comparison

**Treatment & Patient Care Insights**
Treatment type distribution by diagnosis
Average medications per diagnosis
Diagnosis level summary using a matrix table

**Outcome**
This dashboard helps in identifying high-risk patient groups, understanding treatment patterns, and analyzing healthcare trends, which can support better hospital decision-making and patient care strategies.
